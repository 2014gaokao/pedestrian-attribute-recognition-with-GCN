# pedestrian-attribute-recognition-with-GCN
图卷积识别行人特征

prepare rap dataset first

python transform_rap2.py  转换数据格式

python glove.py 产生简单的词向量

python adj.py 生成邻接矩阵

python train.py 训练

python demo.py
